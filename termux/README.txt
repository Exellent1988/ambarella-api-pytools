It is possibile to run the Python scripts from an Android
mobile device, through the Termux app:

* Install the Termux and Termux:Widget Android apps from the F-Droid repo.
* Install the Python package from the Termux terminal (apt install python).
* Copy the Python scripts into a $HOME/bin/sj8pro/ directory.
* Copy shortcuts and icons into the $HOME/.shortcuts/ directory.
* Add a Termux shortcut widget to your desktop to launch each script.
